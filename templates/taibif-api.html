說明 Usage

最後修改：2023/04/25

使用規範：

本 API 採「政府資料開放授權條款－第1版」

回傳格式說明：

  • 回傳格式為 JSON。
  • 查詢網址為 https://portal.taibif.tw/api/{version}/{type}/{mode}?{parameter}，其中：
    • {version}：API 版本，此版本為 v2 。
    • {type}：查詢資料種類，目前提供 occurrence、dataset 與 publisher 。
    • {mode}：限出現紀錄 (occurrence) 資料類型查詢，目前提供 basic_occ（轉譯欄位 + 基本欄位資料回傳）、 detail_occ（轉譯欄位 + 基本欄位 + 常用欄位資料回傳）與 raw_occ（原始資料回傳）。
      各自的差別請看 出現紀錄查詢類型說明。
    • {parameter}：查詢參數
    • 網址範例：
    資料種類{type}範例
    occurrence (basic_occ) https://portal.taibif.tw/api/v2/occurrence/basic_occ?{parameter}=
    occurrence (detail_occ) https://portal.taibif.tw/api/v2/occurrence/detail_occ?{parameter}=
    occurrence (raw_occ) https://portal.taibif.tw/api/v2/occurrence/raw_occ?{parameter}=
    dataset https://portal.taibif.tw/api/v2/dataset?{parameter}=
    publisher https://portal.taibif.tw/api/v2/publisher?{parameter}=

  • 分頁參數說明：

    參數名稱 {parameter} 說明範例
    limit每頁回傳筆數。預設為 100 筆，1000 筆為上限 https://portal.taibif.tw/api/v2/occurrence/basic_occ?rows=100
    https://portal.taibif.tw/api/v2/occurrence/detail_occ?rows=100
    offset 起始回傳資料位置。預設為 0 。 https://portal.taibif.tw/api/v2/occurrence/basic_occ?offset=0
    • 如有建議或額外需求，歡迎聯絡我們。

出現紀錄 Occurrence API

查詢方式：

用途參數名稱 {parameter}查詢網址
取得出現紀錄資料請看回傳欄位說明/api/v2/occurrence/

出現紀錄查詢類型

  • 出現紀錄資料類型查詢 {mode} 說明：
    • 定義名稱描述欄位名稱
    轉譯資料TaiBIF 針對特定欄位資料做清理 (例如統一格式等)共 34 個欄位：taibifOccurrenceID、scientificName、isPreferredName、taxonRank、taicolTaxonId、gbifAcceptedId、scientificNameID、taxonBackbone、eventDate、decimalLongitude、decimalLatitude、geodeticDatum、verbatimSRS、coordinateUncertaintyInMeters、basisOfRecord、selfProduced、license、occurrenceStatus、kingdom、phylum、class、order、family、genus、taxonGroup、country、countryCode、county、year、month、day
    基本資料出現紀錄相關的重要欄位共 18 個欄位：occurrenceID、dataGeneralizations、coordinatePrecision、locality、organismQuantity、organismQuantityType、datasetName、datasetShortName、typeStatus、recordedBy、recordNumber、preservation、collectionID、taibifCreatedDate、taibifModifiedDate、associatedMedia、mediaLicense
    常用資料出現紀錄相關的次要欄位共 4 個欄位：modifiedDate、gbifDatasetID、establishmentMeans、issue
    原始資料資料上傳者所提供的資料
  • 各查詢類型的回傳欄位說明：
    • 類型 {mode}回傳內容回傳欄位
    basic_occ轉譯資料 + 基本資料共回傳 52 個欄位
    detail_occ轉譯資料 + 基本資料 + 常用資料共回傳 56 個欄位
    raw_occ原始資料所有資料上傳者所提供的原始資料

    資料集 Datset API

    查詢方式：

    用途參數名稱 {parameter}查詢網址
    取得資料集資訊datasetName, publisherID, publisherName, author, publicationDate, datasetModifiedDate, gbifDatasetID, license, core/api/v2/dataset/

    查詢參數說明：

    參數名稱 {parameter}說明範例限定搜尋字彙
    datasetName資料集名稱，關鍵字模糊查詢https://portal.taibif.tw/api/v2/dataset?datasetName=海龜
    publisherID同 GBIF 發布單位 IDhttps://portal.taibif.tw/api/v2/dataset?organizationID=7c07cec1-2925-443c-81f1-333e4187bdea
    publisherName發布單位名稱，關鍵字模糊查詢https://portal.taibif.tw/api/v2/dataset?organizationName=Taiwan
    author發布者https://portal.taibif.tw/api/v2/dataset?author=Li
    publicationDate資料集發布日期單一日期：https://portal.taibif.tw/api/v2/dataset?publicationDate=2020-07-30
    兩個日期之間：https://portal.taibif.tw/api/v2/dataset?publicationDate=2013-08-12,2020-07-01
    datasetModifiedDate資料集更新日期https://portal.taibif.tw/api/v2/dataset?datasetModifiedDate=2020-07-30
    gbifDatasetIDGBIF 資料集 IDhttps://portal.taibif.tw/api/v2/dataset?gbifDatasetID=233783dc-e13a-4770-9d24-797590ff8716
    license資料授權格式https://portal.taibif.tw/api/v2/dataset?license=CC0
    https://portal.taibif.tw/api/v2/dataset?license=CC-BY    		CC0, CC-BY, CC-BY-NC, NA
    core資料核心集SAMPLINGEVENT, OCCURRENCE, CHECKLIST, METADATA, OTHER

    其他回傳欄位說明：

    使用 Dataset API 共回傳 12 個欄位。
    回傳欄位說明
    datasetShortName資料集在 IPT 上的 short name
    doi數位物件識別號（Digital Object Identifier）
